# Ticketing

## Description
This is a microservices application built using Node.js and Next.js. The purpose of this application is to demonstrate how to build a microservices architecture using these technologies. The application is designed to be scalable and maintainable.

## Architecture
The application is built using microservices architecture. Each service is a standalone module that communicates with other services using APIs. The services communicate with each other using RESTful APIs.

## Technology Stack
The following technologies are used to build this application:
   . Node.js
   . Next.js
   . Docker
   . Kubernetes
